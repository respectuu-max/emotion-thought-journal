# 마음고요 상담분석실 20260714v55

주별 CSV 자료를 누적해 내담자의 회복 흐름을 시각적으로 피드백하고, 상담 회기에서 측정치 기반 치료 작업을 돕는 상담자용 분석 앱입니다.

## v55 변경 사항

- 각 치료작업 상단에 `측정치의 한계` 안내를 추가했습니다.
- `피드백 안전 확인` 카드를 추가해 수치를 평가가 아니라 공동 탐색으로 다루도록 했습니다.
- `측정치 너머 확인` 카드를 추가해 관계, 환경, 수치심, 기록되지 않은 사건을 함께 확인하도록 했습니다.
- `삶의 회복 연결` 카드를 추가해 문제행동 감소가 실제 삶의 시간, 관계, 돈, 자존감 회복으로 이어지는지 다룹니다.
- `행동화 이후 회복 복귀` 카드를 추가해 재발 후 수치심보다 회복 경로 복귀를 다룹니다.
- `회복 정체성 질문`과 `이번 주 회복 문장 만들기` 카드를 추가했습니다.
- `중독 유형별 초점 상기` 카드를 추가해 도박, 성, 알코올·약물, 게임·인터넷 중독에서 놓치기 쉬운 초점을 상담자가 확인할 수 있게 했습니다.

## v54 변경 사항

- 각 치료작업 상단에 `목적`, `주요 지표`, `기본 치료 방향`, `통찰 질문`을 표시하도록 보강했습니다.
- 상단 9개 지표가 어떤 치료작업에서 사용되는지 확인할 수 있는 `지표-치료작업 연결표`를 추가했습니다.
- `실천행동 조정 필요`를 실패가 아니라 가치 있는 삶을 향한 작은 행동 재설계 신호로 다루도록 7단계 안내를 강화했습니다.
- 7단계에 `실천행동 조정 필요를 다루는 태도`와 `새 경험이 생각을 바꾸는 경로` 카드를 추가했습니다.
- 앱의 기본 방향을 `측정치 기반 피드백 치료`, `변화동기 강화`, `가치기반 실천 습관화`로 더 명확히 드러냈습니다.

## v53 변경 사항

- 상단 9개 지표 중 `정서 주의`와 `인지 주의`에 자료량 배지를 추가했습니다.
  - `신호 없음`
  - `단일 신호`
  - `초기 반복`
  - `반복 신호`
  - `자료 적음`
- 정서·인지 신호가 전체 연쇄 고리 중 몇 건에서 나타났는지 `신호 수/전체 연쇄 수`로 표시합니다.
- 상담자가 정서·인지 지표를 볼 때, 반복 패턴인지 한두 사건의 영향인지 먼저 구분할 수 있도록 설명을 보강했습니다.

## v52 변경 사항

- 상단 9개 지표의 명칭과 설명을 상담자용 임상 언어로 다시 정리했습니다.
- `자료량`에 해석 안정성 배지를 추가했습니다.
  - `해석 주의`
  - `초기 패턴`
  - `반복 패턴 검토 가능`
- `연쇄 고리`에 주된 연쇄 유형을 표시합니다.
  - 행동화 위험형
  - 충동 감내형
  - 감정-충동형
  - 잠복 연쇄형
  - 실천 회복형
- 충동을 행동화하지 않고 견디는 회복 능력을 `충동 감내력`으로 정리했습니다.
- `실천 조정` 표현을 `실천행동 조정 필요`로 바꾸었습니다.
- 회기에서 바로 다룰 변화 자료를 `변화 포착 단서`로 정리하고, 내담자에게 어떤 변화 지점을 피드백할지 설명을 보강했습니다.
- 파국예상과 실제 결과를 비교하는 지표를 `예기불안 검증`으로 정리했습니다.
- 핵심 단어가 한눈에 들어오도록 상단 지표 설명에 강조체와 배지 스타일을 적용했습니다.

## v51에서 유지되는 주요 기능

- 모든 화면 문구의 실천행동 자기효능감 관련 표현은 `성취감`으로 통일되어 있습니다.
- 같은 `id`가 반복되면 `updated_at`이 더 최신인 기록만 분석에 반영합니다.
- `client_alias`가 비어 있는 자료는 첫 파일 단독 분석은 가능하지만, 여러 파일 누적 병합은 차단합니다.
- 누적 자료에 `counselor_full`과 `backup_full`이 함께 들어오면 분석 범위가 넓어질 수 있음을 경고합니다.
- `2. 오늘의 측정 피드백`에는 주별 변화 피드백 그래프와 주별 통계표가 포함되어 있습니다.

## 파일 구성

- `index.html`: 앱 실행 파일
- `styles-20260714-v55.css`: 화면 스타일
- `app-20260714-v55.js`: CSV 읽기, 누적 병합, 주별 통계, 치료 피드백 로직
- `README.md`: 이 설명 파일

## 사용 방법

`index.html`을 브라우저로 열고, 마음고요 관찰과 실천 앱에서 내보낸 CSV를 불러오면 됩니다. 같은 내담자의 주차별 CSV를 순서대로 불러오면 자료가 누적됩니다. 자료를 새로 시작하려면 `자료 제거`를 먼저 누르세요.

## 개인정보 안내

이 앱은 CSV를 서버에 저장하지 않습니다. 다만 브라우저 탭 안에서는 불러온 원본 기록이 분석을 위해 유지됩니다. `전체 자료 저장(CSV)`으로 내보낸 파일에는 민감한 원문 기록이 포함될 수 있으므로 승인된 보관 위치에 저장하세요.
