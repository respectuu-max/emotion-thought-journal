# 마음고요 상담분석실 20260717v66

주별 CSV 자료를 누적해 내담자의 회복 흐름을 시각적으로 피드백하고, 상담 회기에서 측정치 기반 치료 작업을 돕는 상담자용 분석 앱입니다.

## v61 변경 사항

- 활성화된 치료작업 메뉴의 글자색이 흰색으로 보이던 문제를 수정해 검정 계열로 읽히도록 했습니다.
- CSV를 불러온 뒤 지난 한 주 자료를 기반으로 `위기의 한 주`, `인지와 불안이 핵심인 한 주`, `동기 저하`, `감정 감내력·부분변화 강화`, `기본 회기 흐름` 중 하나를 제안합니다.
- 치료작업 순서 바로 아래에 이번 회기 중점 주제, 권장 작업 흐름, 상담자 과업을 표시하는 동적 회기 초점 패널을 추가했습니다.
- 이 자동 평가는 진단이 아니라 상담자가 회기 초반에 확인할 임상적 가설로 표시되도록 안내했습니다.

## v60 변경 사항

- 각 치료작업 상단에 실제 50분 회기에서 상담자가 수행할 핵심 과업을 추가했습니다.
- 왼쪽 패널에 50분 회기 운영구조를 추가해 기본 회기, 시간 부족, 위기, 인지·불안, 동기 저하 상황별 흐름을 볼 수 있게 했습니다.
- 1·3·7·9 필수 단계, 2·6 보조 단계, 4·5·8 선택 단계를 파스텔 톤 배경과 배지로 구분했습니다.
- 치료작업 메뉴와 단계 설명 카드가 같은 필수/보조/선택 색 체계를 사용하도록 정리했습니다.

## v59 변경 사항

- 기존 `충동 감내력`을 `감정 감내력`으로 확장했습니다. 여기서 감정은 중독의 충동과 갈망을 포함하며, 충동은 감정 감내력 창 안에서 다루는 강한 정서/각성 신호로 설명됩니다.
- 9개 치료작업 순서를 `자료와 치료 프레임 확인 → 오늘 마음에 걸린 경험 지도 → 감정 중심 자동 연쇄 재구성 → 개입 분기 판단 → 위기·충동 대처 훈련 → 감정 감내력·부분변화 → 가치기반 작은 실천 설계 → 생각 검토·예기불안 검증 → 회기 통합·다음 주 실험 설계`로 재구성했습니다.
- 과각성(불안·흥분·쾌감), 저각성(지루함·무기력·공허감), 감정 감내력 창을 구분하는 안내 카드를 추가했습니다.
- 작은 실천행동을 새로운 즐거움 찾기가 아니라 불편함을 감내하며 평온하고 안정적인 가치 방향으로 돌아오는 훈련으로 설명했습니다.

## 20260716v65 저장 기능 정리

- `상담기록 요약 저장` / `상담기록·비교 요약 저장(JSON)`: 현재 회기 기록 보관과 다음 회기 비교용 JSON입니다. 원본 CSV 전체가 아니라 분석 요약, 핵심 수치, 상담 메모, 연쇄 요약을 저장합니다.
- `AI 회기준비 자료 저장(MD)`: 최근 1~2주 핵심 정보, 프로젝트 연속성 프롬프트, 이전 회기 맥락 입력란을 마크다운으로 저장합니다. 이번 50분 회기의 초점, 작업 순서, 질문 예시를 구체화할 때 쓰는 준비 자료입니다.
- `AI 상세분석 원자료 저장(TXT)`: 관찰·실천 기록을 길게 정리한 원자료입니다. 회기 프로토콜보다 넓은 자료 검토나 추가 분석에 사용합니다.
- `누적 원본자료 저장(CSV)`: 여러 번 불러온 내담자 CSV를 하나의 원본 CSV로 다시 묶어 저장합니다. 실제 기록 문장이 담긴 민감정보 파일입니다.


## 20260716v65 별칭·AI 연속성 업데이트

- CSV의 `client_alias`가 비어 있으면 상담분석실이 `MG-UNKNOWN-YYYYMMDD-XXXX` 형식의 임시 비식별 자료 코드를 부여합니다.
- 같은 탭에서 이어서 불러오는 빈 별칭 CSV에는 같은 임시 코드를 적용하여 병합 안정성을 높입니다.
- 일부 행만 `client_alias`가 비어 있고 파일 안에 단일 별칭이 있으면, 비어 있는 행에 그 별칭을 보완합니다.
- `AI 회기준비 자료 저장`은 `.md` 마크다운 파일로 저장되며, 상단에 프로젝트 연속성 프롬프트와 이전 회기 맥락 입력란이 포함됩니다.
- AI 프로젝트명은 `마음고요-{client_alias}` 형식으로 자동 제안됩니다.


## 20260716v65 AI 회기 연속성 완성

- 상담사가 `AI 프로젝트명`을 직접 수정할 수 있습니다. 수정된 이름은 AI 회기준비 자료와 AI 회기맥락 누적 요약에 반영됩니다.
- `이전 AI 회기맥락 불러오기` 기능을 추가했습니다. 지난 회기에 저장한 `*_AI회기맥락누적요약.md` 파일을 불러오면 새 AI 회기준비 자료에 자동 포함됩니다.
- `AI 회기맥락 누적 요약 저장(MD)` 기능을 추가했습니다. 상담자가 채택한 AI 제안, 보류한 가설, 다음 회기 확인 질문, 다음 AI에게 전달할 맥락을 구조화해 저장합니다.
