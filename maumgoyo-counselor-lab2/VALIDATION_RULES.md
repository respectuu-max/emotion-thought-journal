# 표준 근거자료 사전검증 규칙

## 오류: 저장을 차단하는 항목

- 스키마가 `maeumgoyo_ai_evidence_v1`이 아님
- 패키지 ID 없음
- AI 근거자료에 사용할 수 있는 CSV 행이 0건
- 사용 가능 행 수와 `source_evidence` 건수가 일치하지 않음
- `payload_json` 파싱 실패
- 근거 ID 누락
- 근거 ID 중복
- 허용되지 않은 출처 코드

## 경고: 상담자가 검토한 후 저장할 수 있는 항목

- 임시 UNKNOWN 비식별 코드
- 날짜가 있는 일상 기록 부족
- 상담 회기일 또는 참여 상태 미입력
- 상담자가 직접 확인한 사실 미입력
- 관찰기간 내부 무기록일
- 기록일보다 2일 이상 늦은 지연 작성
- 이메일·전화번호·주민등록번호·웹주소 형식 후보
- 스키마 또는 공유 모드 조건으로 제외된 행

경고가 없더라도 실명, 기관명, 직장·학교, 구체적 장소와 맥락상 간접식별자는 사람이 직접 확인해야 한다.

## 감사 기록(gratitude_others/gratitude_self)은 오류·경고 대상이 아님

`daily_checkin`의 감사 필드는 선택 입력이므로, 값이 없거나 빈 배열이어도 오류나 경고를 발생시키지 않는다. 문자열(v100)과 배열(v101) 중 어떤 형태로 와도 정규화 후 통과시킨다. 건수가 적거나 0건인 기간이 있어도 사전검증 상태(`valid`/`valid_with_warnings`/`invalid`)에 영향을 주지 않는다.

## 프로필별 추가 준비조건

| 프로필 | 추가 검사 |
|---|---|
| A | 분석할 자동 연쇄 유무 |
| B | 자동 연쇄와 상세분석 자료 유무 |
| C | 이전 맥락 또는 상담자 메모 유무 |
| D | observation 자기관찰 근거 유무 |
| E | observation 자기관찰 근거 유무 |
| F | 회기일, 참여 상태, 상담자 직접 확인 사실 필수; 과제·확인 한계 권고 |
| G | 최소 21일 자료범위; 주차 간 비교자료 권고 |
| H | 회기일, 참여 상태, 상담자 직접 확인 사실, 확인 범위와 한계 필수 |

## 상태값

- `valid`: 자동점검 오류·경고 없음
- `valid_with_warnings`: 오류는 없지만 상담자 검토가 필요한 경고 존재
- `invalid`: 하나 이상의 오류가 있어 표준 JSON 저장 불가

사전검증 통과는 자료의 진실성, 치료효과, 반성의 진정성 또는 재범위험 감소를 보증하지 않는다.
