# 마음고요 상담분석실 v11

## 실행 방법

1. 이전 폴더와 다른 새 폴더에 압축을 풉니다.
2. 기존에 열어 둔 앱 창을 닫고, 새 폴더의 `index.html`을 최신 웹 브라우저에서 엽니다.
3. CSV 파일을 선택하거나 화면 위로 끌어 놓습니다.
4. 화면의 8단계 메뉴에서 측정치 기반 상담 피드백을 검토합니다.

`현재 파일 새로고침` 버튼은 이미 열려 있는 같은 버전의 파일만 다시 읽습니다. 새 기능은 새 ZIP 파일을 풀고 그 안의 `index.html`을 열어 적용합니다. 화면 상단 배지가 `20260710v12`이면 업데이트가 적용된 것입니다.

## 자료 보호 안내

- 이 앱은 불러온 CSV를 브라우저 메모리에서만 처리합니다.
- 요약 복사, 인쇄, 요약 저장 결과에는 실제 기록 문장과 상담 메모가 포함될 수 있습니다.
- 승인된 기기와 암호화된 보관 위치에서만 사용하고, 공유 모드·민감정보 제외 상태를 상담자가 확인하세요.
- 한 번에 한 명의 내담자와 하나의 공유 모드로 구성된 CSV만 분석할 수 있습니다.

## CSV 형식

권장 형식은 `maeumgoyo_compact_v2`이며, `schema_version`, `record_type`, `id`, `payload_json` 열이 필요합니다. 실천 기록을 관찰 연쇄와 연결하려면 `payload_json`에 `observation_id`, `chain_id` 또는 `episode_id`를 포함하세요.

## 재발 조기경보

- 정서적 재발: 정서 점수, 부정정서 표현, 실천 저수행을 함께 확인합니다.
- 인지적 재발: 충동 점수와 문제행동에 접근·합리화하는 생각을 함께 확인합니다.
- 두 종류의 신호는 순서와 관계없이 같은 기록에서 함께 경고될 수 있습니다.
- 행동 기록이 있으면 비난 없이 즉시 인정, 신뢰할 사람·상담자와 연결, 회복계획 재검토를 안내합니다.
- 이 경고는 진단이 아니라 상담 공백일의 조기 자기관리와 도움요청을 돕기 위한 안내입니다.

## 실천 수행도 계산

실천 과제의 평균은 기록된 점수만 평균내지 않습니다. `planned_days_per_week × planned_times_per_day`로 계산한 약속 총 횟수를 분모로 사용합니다. 앱은 계획 대비 완료율, 계획 반영 평균 실천점수, 기록률을 함께 표시합니다.
