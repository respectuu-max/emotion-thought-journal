마음고요 관찰과 실천 배포용 폴더

이 폴더 전체를 Netlify, Cloudflare Pages, GitHub Pages 같은 정적 호스팅에 올리면 됩니다.

가장 쉬운 방법:
1. Netlify에 로그인합니다.
2. 새 사이트 배포 화면에서 이 폴더(maeumgoyo-observe-practice)를 끌어다 놓습니다.
3. 생성된 https 주소를 복사합니다.
4. 네이버 블로그나 Google Sites에 "마음고요 관찰과 실천 열기" 버튼 또는 QR 코드로 연결합니다.

주의:
- 내담자에게 HTML 파일을 직접 내려받게 하는 방식보다 https 주소 접속 방식이 안정적입니다.
- 기록은 서버가 아니라 내담자의 스마트폰/브라우저 저장공간에 저장됩니다.
- 공유 전 미리보기에서 상담자용 상세, 상담자용 요약, 가족 회복요약을 구분해 확인할 수 있습니다.
- 공유 메뉴의 CSV 가져오기로 이 앱에서 저장한 CSV 기록을 다시 불러올 수 있습니다.
- 스마트폰에서 예전 화면이 계속 보이면 보호설정의 "앱 업데이트 새로 적용" 버튼을 누르세요.
- 앱은 응급 대응 도구가 아니며, 위기 상황에서는 즉시 사람에게 도움을 요청해야 합니다.
