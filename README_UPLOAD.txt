감정사고 관찰일지 배포용 폴더

이 폴더 전체를 Netlify, Cloudflare Pages, GitHub Pages 같은 정적 호스팅에 올리면 됩니다.

가장 쉬운 방법:
1. Netlify에 로그인합니다.
2. 새 사이트 배포 화면에서 이 폴더(emotion-thought-journal-deploy)를 드래그해 올립니다.
3. 생성된 https 주소를 복사합니다.
4. 네이버 블로그나 Google Sites에 "감정사고 관찰일지 열기" 버튼 또는 QR코드로 연결합니다.

주의:
- 내담자에게 HTML 파일을 다운로드시키는 방식이 아니라, 생성된 https 주소로 접속하게 하세요.
- 스마트폰 홈 화면 추가와 오프라인 실행은 https 주소에서 가장 안정적으로 작동합니다.
- 기록은 서버가 아니라 내담자 스마트폰 브라우저 저장공간에 저장됩니다.
