# GitHub Pages 업로드용 파일

이 폴더 안의 파일 전체를 GitHub 저장소 루트에 올리세요.

필수 파일:
- index.html
- manifest.webmanifest
- service-worker.js
- app-icon-192.png
- app-icon-512.png
- maeumgoyo_logo_large_subtitle.png
- .nojekyll

GitHub Pages 설정:
1. GitHub 저장소 생성
2. 위 파일들을 저장소 루트에 업로드
3. Settings > Pages
4. Source: Deploy from a branch
5. Branch: main / root 선택
6. 표시되는 Pages 주소를 내담자에게 공유

주의:
- Pages 주소가 바뀌면 내담자 스마트폰에 저장된 기존 기록이 이어지지 않을 수 있습니다.
- 저장소 이름과 Pages 주소는 운영 시작 후 되도록 바꾸지 마세요.
